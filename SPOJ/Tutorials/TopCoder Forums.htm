Let sum(i) be the sum of numbers 0 ... i, then the sum of numbers in the interval j ... i is sum(i) - sum(j - 1). So for each index i you try to find an index j such that sum(i) - sum(j - 1) is maximum, this is the same as minimizing sum(j - 1). You can therefore process numbers from left to right, and update the minimum...
Counting is done the same way.
Hope it helps!

it has a simple recursive solution:

if(maxim(i-1)+a[i]==a[i]){
maxim(i)=maxim(i-1)+a[i]
ways(i)=ways(i-1)+1;
}
else if(maxim(i-1)+a[i]>a[i]){
maxim(i)=maximum(i-1)+a[i];
ways(i)=ways(i-1);
}
else{
maxim(i)=a[i]
ways(i)=1
}
simply find maxim(i) among all i's and sum corresponding ways(i)

here:
a[i] is given array of integers
maxim(i) is the maximum sum of any contiguous subsequence ending at position i.
ways(i)=ways to get this sum

		for(int i=0;i<n;i++)
			scanf("%d",&(a[i]));
		if(n==1){
			printf("%d %d\n",a[0],1);
			continue;
		}
		int b[n];
		long long int c[n];
		b[0]=a[0];
		c[0]=1;
		int mx=-1000000000;
		for(int i=1;i<n;i++){
			int tmp=b[i-1]+a[i];
			if(tmp==a[i]){
				b[i]=tmp;
				c[i]=c[i-1]+1;
			}
			else{
				if(a[i]>tmp){
					b[i]=a[i];
					c[i]=1;
				}
				else{
					b[i]=tmp;
					c[i]=c[i-1];
				}
			}
			if(b[i]>mx)
				mx=b[i];
 
		}
		long long int ret=0;
		for(int i=0;i<n;i++){
			if(b[i]==mx)
				ret+=c[i];
		}
		printf("%d %lld\n",mx,ret);
	}

I did exactly like Sarkin . No i/o optimizations needed though. Here is the main part of my code, after reading an array A.
* rmx is max contiguous sum and rct is its count
* mn is min prefix sum and mnct is its count
rmx = -inf;
mn = 0; mnct = 1;
 
for(i=sum=0;i<n;i++)
{
	sum += A[i];
			
	if( sum - mn > rmx) rmx = sum-mn ,  rct = mnct;
	else if( sum - mn == rmx ) rct += mnct;
 
	if( sum < mn ) mn = sum ,  mnct = 1;
	else if( sum == mn ) mnct++;
}
