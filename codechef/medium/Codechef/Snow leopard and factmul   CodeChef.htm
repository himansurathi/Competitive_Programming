Snow leopard and factmul

Problem code: MULFACT
  • My Submissions
  • All Submissions

All submissions for this problem are available.

Snow leopard is given a different task this time. He has to find the product of factorials, but he is facing a lot of problems in finding the optimal code. Please can you help him out.

fact(n) returns an integer of product of all factorials upto n,i.e., 1! * 2! * ..... (n-1)! * n!

Since the product can be very large, print fact(n) % 329885391853.
 

Input

First line contains an integer ,N, the number of test cases.

Next N lines contain an integer(m).
 

Output

 N lines containing fact(m) % 329885391853

Constraints

1 < n <= 10

0 < m <= 1000000000
 

Example

Input:
2

4

5

Output:
288

34560
